# Kafka & Node.js Microservices

A setup for learning **Kafka partitions**, **multi-broker clusters**, and **Node.js microservices**.

### Run Kafka

```bash
cd kafka
docker compose up
```

### Run Services

```bash
node --watch index
```

### Run Client

```bash
npm run dev
```
